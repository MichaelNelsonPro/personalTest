# 音频系统使用说明

## 背景音乐 (BGM)

### 添加 BGM
1. 将你的背景音乐文件（MP3 格式）放入 `public/` 目录
2. 命名为 `bgm.mp3`
3. 刷新页面后点击右下角的设置按钮开启 BGM

### 推荐音乐风格
- EVA 风格：紧张、科幻、管弦乐
- 可以去网上搜索 "EVA BGM" 或 "NERV 风格音乐"
- 注意：请使用无版权音乐或自己创作

## 语音播报

### 功能说明
- 使用浏览器内置的语音合成 API (TTS)
- 自动播报系统状态（日语女声）
- 每 15 秒播报一次
- 当系统有警告时也会立即播报

### 播报内容示例
```
"CPU使用率45パーセント、全システム正常"
"警告、メモリ圧迫、85パーセント"
"バッテリー残量20パーセント、充電してください"
```

### 使用明日香真人语音（进阶）
如果你想使用真正的明日香语音，需要：

1. **使用 AI 语音克隆工具**（如：ElevenLabs、VITS）
   - 准备明日香的音频样本
   - 训练或克隆语音模型
   - 生成系统状态播报音频

2. **预录制音频文件**
   - 录制/生成常用的播报语句
   - 放入 `public/voice/` 目录
   - 修改 `useAudioSystem.ts` 使用预录音频而非 TTS

### 浏览器兼容性
- Chrome/Edge：✅ 完整支持
- Firefox：✅ 支持
- Safari：⚠️ 部分支持

## 音量控制

点击右下角的 ⚙️ 设置按钮：
- **BGM**：开关和音量调节
- **VOICE REPORT**：语音播报开关和音量调节

## 注意事项

1. **自动播放策略**：现代浏览器可能会阻止自动播放音频，需要用户先与页面交互
2. **版权问题**：请确保使用的音乐没有版权限制
3. **语音质量**：浏览器 TTS 的日语声音因系统而异，Windows/Mac 的语音包不同
